Installation

Server Requirements

  • PHP >= 8.2
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension

Via Composer

composer create-project tinymvc/tinymvc myapp

Or, Clone Repository

git clone https://github.com/tinymvc/tinymvc.git
cd tinymvc
composer install
Application Key: After installation, set your application key in env.php. Generate it with: 
php spark key:generate

You can skip it, for the first time when you use composer to create-project.

Serving Your Application

php spark serve
