Installation

Via Composer

composer create-project tinymvc/tinymvc myapp

Clone Repository

git clone https://github.com/tinymvc/tinymvc.git
cd tinymvc
composer install
Application Key: After installation, set your application key in env.php. Generate it with: php spark key:generate

Server Requirements

  • PHP >= 8.0
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension

Serving Your Application

php spark serve
